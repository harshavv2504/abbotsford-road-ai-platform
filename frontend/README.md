# Abbotsford Road Coffee Specialists - AI Chatbot & CRM

Frontend application for Abbotsford Road Coffee Specialists with AI chatbot and CRM functionality.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Features

- AI-powered chatbot with voice input
- CRM system with Kanban board
- Lead management from chatbot conversations
- User authentication

## Note

The chatbot currently uses mock responses. Backend integration will provide full AI capabilities.
